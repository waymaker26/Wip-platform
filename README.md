
# Waymaker Intelligence Platform — V0.1 Foundation

This is the first working internal prototype for the `wip-platform` repository.

## Included
- Command Center dashboard
- Property list builder and Buy Box filters
- Search by address, owner, or parcel number
- ZIP, property type, strategy, score, value, and absentee-owner filters
- CSV export
- Property detail view with opportunity scores
- Project pipeline with editable stages
- Mobile-friendly layout
- Browser persistence using localStorage

## Publish with Vercel
1. Upload all files directly to `Waymaker-Building-Co/wip-platform`.
2. Import the repository into Vercel as a new project.
3. Framework preset: **Other**.
4. No build command is required.
5. Add `app.waymakerbuildingco.com` later as the custom domain.

## Important
This release uses sample data to validate the workflow. Live parcel ingestion,
authentication, Supabase, mapping, and shared cloud data are the next milestones.
